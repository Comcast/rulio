### /loc/events/ingest

Submit an event for processing

1. `location`: The target location
2. `event`: The event in JSON
3. `externalFactsTimeout`: Optional timeout; syntax is from 'http://golang.org/pkg/time/#ParseDuration'


Example request:

```Javascript

    "person": "homer"
  },
  "location": "here",
  "uri": "/loc/events/ingest"
```
