This API is a simple HTTP rendering of the primtive API, which can be packaged up as a REST API, 0MQ API, Web sockets API, etc.

This documentation was generated by rulesapi/rulesapi.

Index

  1. /version Report the version of the endpoint
  2. /sys/util/nowsecs Report the time that the endpoint thinks it is
  3. /sys/util/match Match a fact against a pattern
  4. /sys/util/js Evaluate some in-process Javascript
  5. /sys/util/batch Evaluate a batch of requests
  6. /sys/cachedlocations Get all cached locations
  7. /loc/admin/size Gets the number of facts (including rules) in the location
  8. /loc/facts/add Add a fact to the given location
  9. /loc/facts/search Search (internal) facts at given location
  10. /loc/facts/query Query facts at given location
  11. /loc/rules/add Add a new rule to the given location
  12. /loc/rules/list Get the known rules
  13. /loc/events/ingest Submit an event for processing
  14. /loc/events/retry Retry event processing
  15. /sys/admin/timers/names Get an array of known timers
  16. /sys/admin/timers/get Get a timer
  17. /loc/rules/disable Disable a rule
  18. /loc/rules/enable Enbale a rule
  19. /loc/rules/enabled Report whether a rule is enabled
  20. /loc/rules/rem Remove a rule from the given location
  21. /loc/util/js Evaluate some in-process Javascript within a location
  22. /loc/admin/create Create a location
  23. /loc/admin/delete Delete a location
  24. /loc/admin/stats Get basic stats for the given location
  25. /loc/admin/clear Clear all (internal) state at the given location
  26. /sys/runtime Get a huge pile of Go runtime stats
  27. /sys/admin/pending Get or set the max number of pending requests
  28. /sys/admin/connstates See the connection state counts
  29. /sys/admin/gc Run the garbage collector
  30. /sys/admin/freemem Maybe return memory to the OS
  31. /sys/admin/gcpercent Set the GC trigger threshold
  32. /sys/stats Get basic system metrics
  33. /health/shallow Ping
  34. /health/deep Somewhat deep health check
  35. /health/deeper Deeper health check, which checks storage
